# 주사위 게임
## 진행 방법
* 주사위 게임
* 이전 미션과 동일

###기능 요구사항
* 입력인원을 입력받는다 최대 4명
* 참가자의 이름을 입력받는다.
* 참가자의 이름은 최대 5자리이고 영문만 가능하다.
* 참가자별 주사위 2개를 굴린다.
* 주사위의 합이 높은사람이 우승이다.
* 우승자는 여러명이 가능하다.

### 실행화면
#### 참여 인원은 몇명인가요?(최대4명)
#### 3
#### 참가자의 이름을 입력하세요 :
#### aaa
#### 참가자의 이름을 입력하세요 :
#### bbb
#### 참가자의 이름을 입력하세요 :
#### ccc
#### 실행결과
#### aaa : 5 - 5 : 10
#### bbb : 2 - 6 : 8
#### ccc : 4 - 6 : 10
#### 우승자는 aaa, bbb 입니다.
